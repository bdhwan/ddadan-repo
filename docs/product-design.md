# DDADAN 서비스 설계 문서

## 1. 서비스 개요

DDADAN은 프랜차이즈가 아닌 소상공인 매장에서 TV 또는 모니터를 디지털 메뉴판, 광고판으로 사용할 수 있게 해주는 SaaS + 하드웨어 서비스다.

고객은 라즈베리파이를 구매해 매장 모니터에 연결하고, 웹 관리도구에서 메뉴, 이미지, 영상, 공지사항을 수정하면 매장 화면에 전체화면으로 반영된다.

핵심 가치는 다음과 같다.

- 설치가 쉽다
- 월 구독 비용이 부담되지 않는다
- 인쇄물 교체 없이 화면을 즉시 바꿀 수 있다
- 영상, 이미지, 웹 레이아웃을 하나의 플레이어에서 운영할 수 있다
- 소상공인에게 필요한 수준으로 단순하다

## 2. 타겟 고객

### 2.1 1차 타겟

- 개인 카페
- 소형 음식점
- 분식집
- 베이커리
- 미용실, 네일샵
- 무인매장

### 2.2 고객 특성

- 본사 시스템이 없는 독립 매장
- 키오스크나 대형 사이니지 시스템은 부담스러움
- 가격, 메뉴, 홍보 이미지를 수시로 바꾸고 싶음
- 복잡한 CMS보다 쉬운 관리 화면을 선호함

---

## 3. 고객 문제

소상공인 매장 운영자는 보통 아래 문제를 겪는다.

- 메뉴판 인쇄물을 자주 다시 제작해야 함
- 가격 변경이 생겨도 즉시 반영하기 어려움
- TV/모니터는 있어도 메뉴판용으로 활용하기 어려움
- 기존 디지털 사이니지 솔루션은 비싸고 복잡함
- 영상, 이미지, 공지, 이벤트 배너를 손쉽게 교체할 수 없음

---

## 4. 제품 비전

DDADAN의 비전은 다음과 같다.

> 소상공인 누구나 매장 화면을 쉽게 디지털 메뉴판과 광고판으로 바꿀 수 있게 만든다.

기업형 디지털 사이니지가 아니라, 사장님이 스스로 설치하고 운영할 수 있는 가장 쉬운 매장용 스마트 화면 솔루션을 목표로 한다.

## 5. 제품 구성

DDADAN은 세 가지 구성요소로 나뉜다.

### 5.1 하드웨어

- Raspberry Pi 본체
- 전원 어댑터
- HDMI 연결
- 케이스 포함 가능
- 매장 TV/모니터에 연결
- 부팅 후 자동 실행

### 5.2 클라이언트 플레이어

- 라즈베리파이에 연결된 모니터에서 전체화면 실행
- Angular 기반 웹앱 사용
- 이미지, 영상, 웹 기반 메뉴 레이아웃 재생
- 오프라인 캐시 지원
- 부팅 시 자동 실행
- 장애 시 자동 재실행

### 5.3 관리 SaaS

- 웹 기반 관리 도구
- 메뉴, 가격, 공지, 홍보 콘텐츠 업로드
- 디바이스 등록 및 화면 할당
- 플레이리스트 편집
- 스케줄 관리
- 원격 상태 확인

## 7. 핵심 시나리오

### 7.1 첫 설치

1. 고객이 DDADAN 기기를 구매한다.
2. 매장 모니터에 HDMI로 연결한다.
3. 전원을 켜면 기기가 자동 부팅된다.
4. 디바이스 등록 코드 또는 QR이 표시된다.
5. 사장님이 관리 페이지에서 등록 코드를 입력한다.
6. 기기가 매장 계정에 연결된다.
7. 선택한 플레이리스트가 화면에 표시된다.

### 7.2 메뉴 변경

1. 사장님이 관리자 페이지 로그인
2. 메뉴명 또는 가격 수정
3. 저장 후 즉시 배포
4. 라즈베리파이 플레이어가 변경 내용 수신
5. 매장 화면에 자동 반영

### 7.3 홍보 콘텐츠 교체

1. 이미지 또는 영상 업로드
2. 재생 순서와 노출 시간 설정
3. 특정 시간대에만 노출하도록 스케줄 설정
4. 저장 후 각 화면에 자동 반영

## 11. 기술 아키텍처

모노레포 기준 구조:

- `apps/ddadan-api-server`
  - NestJS API 서버
  - 인증, 매장, 디바이스, 콘텐츠, 플레이리스트 API
  - BullMQ 기반 비동기 작업 처리
- `apps/ddadan-client-app`
  - Angular 기반 전체화면 플레이어
  - 라즈베리파이 브라우저 kiosk 모드에서 실행
- `services/ddadan-service-pi`
  - Raspberry Pi 로컬 서비스
  - 기기 등록, kiosk 실행, 상태 보고, 자동 복구

### 11.1 서버 역할

- 사용자 인증
- 매장 및 디바이스 관리
- 콘텐츠 메타데이터 저장
- 디바이스별 플레이리스트 배포
- heartbeat 수집
- 원격 명령 큐 처리

### 11.2 클라이언트 앱 역할

- 전체화면 렌더링
- 콘텐츠 재생
- 재생 순서 제어
- 서버 설정 반영
- 로컬 캐시로 오프라인 대응

### 11.3 Pi 서비스 역할

- 부팅 시 Chromium kiosk 모드 실행
- 플레이어 앱 자동 시작
- 프로세스 죽음 감지 및 재실행
- 서버와 주기적 통신
- 장치 정보 수집

## 14. 차별화 포인트

DDADAN은 기존 디지털 사이니지와 다르게 아래 포지셔닝을 가진다.

- 기업형이 아니라 소상공인형
- 복잡한 CMS가 아니라 간단한 관리도구
- 고가 장비가 아니라 Raspberry Pi 기반 저비용
- 메뉴판 중심 UX
- 쉽게 사고 쉽게 설치 가능

핵심 메시지:

- TV를 스마트 메뉴판으로 바꾸세요
- 가격표와 이벤트 화면을 1분 만에 수정하세요
- 가장 쉬운 소상공인용 디지털 사이니지

## 18. 결론

DDADAN은 소상공인 매장에 최적화된 디지털 메뉴판/광고판 서비스다.

핵심은 복잡한 기업용 솔루션이 아니라, 사장님이 직접 구매하고 설치하고 수정할 수 있는 단순하고 저렴한 서비스 경험이다.

초기에는 메뉴판과 홍보판에 집중하고, 이후 디바이스 운영 자동화와 매장 관리 기능을 강화하면서 SaaS + 하드웨어 결합 서비스로 확장한다.
